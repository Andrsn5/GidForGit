1)git config --global user.name "Andrey Solovyh" (авторизация)
2)git config --global user.email "andrejsolovyh6@gmail.com" (тоже самое)
3) cd git_Idea - выбор папки 
4) dir - просмотр директорий 
5) git remote add origin https://github.com/Andrsn5/TestForMe.git (связь с сервером и добавление origin)
6) git clone https://github.com/BcemXaxa/Crammer.git - скачать папку с сервера (сначала выбрать папку для авторизации)
7) git status - проверяем коммит
8)git add . - добавление всех файлов 
9)git commit -m "added crammer" - добавление коммита 
10)git log - проверяем все коммиты
11)git checkout 93b3a597c824042019a59b15
0b59ed507f96b692 проверяем определенный коммит
12)git push origin master - загружаем все на сервер
13)git pull origin master - догружаем все изменения из сервера 
14)git branch - проверяем где голова 
15) git branch master - создаем новую ветку 
16)git checkout feature - переключаем голову
17)git merge feature - слияние веток
